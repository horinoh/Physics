## 慣性テンソル

### 並行軸の定理 (Parallel Axis Theorem)
- 任意の軸回りの慣性モーメントを求める
- モデル中心と重心がずれている場合など、重心からずれた軸で回転させる場合に使用
- $I_g$ は重心を通る軸回りの慣性モーメント、M は質量、d は重心から軸までの距離

$
    I = I_g + M d^2
$

### 形状毎の慣性モーメント

- 球

$
    I_g = \frac{2}{5} M
    \begin{pmatrix}
    R^2 & 0   & 0   \\
    0   & R^2 & 0   \\
    0   & 0   & R^2
    \end{pmatrix}
$

- 箱

$
    I_g = \frac{1}{12} M 
    \begin{pmatrix} 
    (H^2 + D^2) & 0 & 0 \\ 
    0 & (W^2 + D^2) & 0 \\ 
    0 & 0 & (W^2 + H^2)
    \end{pmatrix}
$

- シリンダー

$
    I_g = \frac{1}{12} M
    \begin{pmatrix}
    (3R^2 + H^2) & 0 & 0 \\
    0 & (3R^2 + H^2) & 0 \\
    0 & 0 & R^2
    \end{pmatrix}
$
