# シーン

## ブロードフェーズ
- SAP (Seep And Prune) で潜在的衝突ペアを作成

## ナローフェーズ
- 潜在的衝突ペアに対して、衝突するであろう直前まで時間を進め、GJK + EPA で判定
- 衝突していれば、衝突情報を収集
    - 衝突時間が 0 の場合は静的衝突 (制約リストへ)
    - 衝突時間が 非0 の場合はは動的衝突 (衝突リストへ)
- 衝突しないままデルタ時間を使い切れば終了

## 制約解決

## シミュレーションを進める
- 衝突リストから、次の TOI (Time Of Impact) までシミュレーションを進める
- 衝突による力積の適用
- 残りのデルタ時間を使い切るまでシミュレーションを進める
